# Account

Setup is done entirely via the [integration UI](https://my.home-assistant.io/redirect/config_flow_start/?domain=smol).

## Name

This is a name or nickname to help identify the account. This is to support multiple accounts. All associated sensors will have this name in the entity name.

## Username

The username or email address used to access your Smol account.

## Password

The password for your Smol account.
